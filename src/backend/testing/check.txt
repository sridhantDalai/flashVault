This application enables users to transfer files quickly using a 
unique locker room system.
A user logs in once using their Google account. After logging in, 
they are assigned a unique locker room number. 
This locker room acts as a temporary space where files can be uploaded.
On any other computer, the user can simply enter the locker room number 
to access the upload interface. 
From there, they can send files, documents, or any 
type of data into that locker room.
Each access session allows only one operation (one batch of file uploads).
 Once the operation is completed, the system automatically logs out the session.
To perform another transfer, the user can re-enter the locker room 
number and repeat the process, 
again limited to a single operation followed by automatic logout.
This system focuses on:
Simple access using a unique locker room number
One-time operation per session
Automatic logout after each use
